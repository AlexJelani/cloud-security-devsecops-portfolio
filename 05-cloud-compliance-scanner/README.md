# Project 5: Cloud Misconfiguration Scanner & Compliance Checker

## 🎯 Objective
Build automated cloud security scanner to detect misconfigurations and compliance violations across AWS and Azure.

## 🏗️ Architecture
- Python-based scanner
- AWS/Azure API integration
- Prowler/ScoutSuite integration
- GitHub Actions automation
- Automated reporting

## 🔍 Detection Capabilities
- Public S3 buckets
- Unencrypted resources
- IAM misconfigurations
- Network exposure
- CIS benchmark violations
- Compliance gaps

## 📋 Prerequisites
- Python 3.9+
- AWS/Azure CLI
- API credentials
- GitHub account

$0-10/month (API calls only)
