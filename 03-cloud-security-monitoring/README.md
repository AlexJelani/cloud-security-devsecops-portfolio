# Project 3: Cloud-Native Security Monitoring & Response

## 🎯 Objective
Deploy cloud-native security monitoring with automated threat detection and response using GuardDuty/Defender and automated remediation.

## 🏗️ Architecture
- EKS/AKS cluster with sample workloads
- GuardDuty (AWS) or Defender for Cloud (Azure)
- Lambda/Azure Functions for automated response
- CloudWatch/Azure Monitor for alerting
- Sentinel integration for SIEM

## 🔒 Security Capabilities
- Real-time threat detection
- Automated incident response
- Crypto-mining detection
- Unauthorized access alerts
- Data exfiltration monitoring

## 📋 Prerequisites
- AWS/Azure account
- Terraform
- kubectl
- Basic Kubernetes knowledge

## 💰 Cost Estimate
$100-200/month (EKS/AKS clusters)
Use Minikube locally to reduce costs
