# Project 4: Kubernetes Hardening & Container Security

## 🎯 Objective
Demonstrate Kubernetes security hardening from vulnerable baseline to production-ready secure deployment.

## 🏗️ Architecture
- Kubernetes cluster (Minikube/EKS)
- Vulnerable app deployment (Kubernetes Goat)
- Security controls implementation
- Before/after vulnerability metrics

## 🔒 Security Controls
- Pod Security Admission
- Network Policies
- RBAC (Role-Based Access Control)
- Image signing with Cosign
- Trivy container scanning
- Runtime security monitoring

## 📋 Prerequisites
- Minikube or EKS access
- kubectl
- Docker
- Helm

$0-50/month (Minikube local or minimal EKS)
